Workshop: Unity/ Projection mapping

I had to play around with Unity to make a game which has VR elements to it. I also had to use madmapper to manipulate projections. On this page you'll find screenshots of the process and a video for the assignments I had to put on the website.

Unity assignment

For this assignment I had to make a interactive game with the help of unity. The interactive element is that every time i walk towards the zombie, it lights up in flames. It only lights up in a small flame though. It was made so that the zombie falls over when it tries to attack the damsel in distress. In the photo you can see the unity develop screen and in the video you can see a demo of the game.

Madmapper assignment

For this assignment I had to project an animation via madmapper and manipulate the projection. In the photo you can see how I achieved this in the madmapper menu.
