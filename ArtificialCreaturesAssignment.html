Artificial Creatures Assignment

A printer needs to get a print assignment from a user’s laptop to do its primary function which is printing. It needs an input from a user to be able to print. Another function for the printer is that it can clean it’s printhead, but this function needs to be implemented by the user itself. You can add a sensor to the printer where it could detect if the printhead is dirty. This would tell the printer to clean the printhead. This would be a time efficient solution to having a working printer without the hassle of having the printer clean itself when you really need to print.
