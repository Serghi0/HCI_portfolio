Workshop: Computer Vision

I had to some programming for the Computer Vision workshop. On this page you'll find the codes I used in Processing for the assignments I had to put on the website. You will also find images or videos demonstrating how the codes work. Furthermore, you'll get a description of what the assignment was about and how the assignment was accomplished. There is also a bit on Computer Vision in there.

Assignment 1

See the Pen Processing 1 by Sergh (@Serghi0) on CodePen.

For this assigment I had to make ellipses at various locations, with various colors, various widths and heights and with and without borders.

Assignment 2

See the Pen processing 2 by Sergh (@Serghi0) on CodePen.

For this assignment I had to write a program that draws at least four different shapes (which were not circles) in various colors and at various locations.

Assignment 3

See the Pen Processing 3 by Sergh (@Serghi0) on CodePen.

For this assignment I had to write a program that draws at least two different shapes, with various colors and make them interactive.

Assignment CV1

See the Pen Processing CV1 by Sergh (@Serghi0) on CodePen.

For this assignment I had to draw a circle around a recognized face.

Assignment CV2

See the Pen Processing CV2 by Sergh (@Serghi0) on CodePen.

For this assignment I had to adjust the code so that every recognized face gets an 'anonymous' black bar in front of his eyes or a red clown nose instead of his/her nose. I chose the former over the latter.

Assignment Bonus

See the Pen Processing CV Bonus by Sergh (@Serghi0) on CodePen.

For this assignment I had to adjust the code so that every recognized image is enhanced by an image e.g. give a recognized face funny glasses or a hat. I chose to give the recognized face a pair of glasses to wear.

Assignment CV3

See the Pen Processing CV3 by Sergh (@Serghi0) on CodePen.

For this assignment I had to adjust the OpenCVBrightestPointDetectionLiveCam code so that something cool would happen. The OpenCVBrightestPointDetectionLiveCam code made sure to put a red dot at the brightest point detected on the live cam feed. I adjusted this so that a rage meme face would be put at the brightest point detected.
